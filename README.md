# BMP-file-processing
Project that handles images in the BMP format as binary files. An in-terminal program that:
-shows particular information about the image
-adds a grayscale filter
-adds an inverted colours filter
-adds a blur filter. 

For every filter added, a new image is created or the initial one can be overwritten. 
In addition, it can:
-crop an image to the specified dimensions
-brighten the image by a specified percentage
-rotate the image by 90 or 180 degrees.
